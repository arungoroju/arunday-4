# arunday‑4 — JavaScript & Web Practice Exercises

This repository contains a set of **HTML and JavaScript practice exercises** created as part of my web development and JavaScript learning sessions. Each file demonstrates a different concept or feature implementation in client‑side web programming.

These practice files help build foundational skills in web interfaces, DOM manipulation, JavaScript functions, and browser interaction.

---

## Project Contents

The repository includes:

- **Anonymous.html / Anonymous.js**  
  Demonstrates JavaScript usage with an anonymous function or interaction on a webpage.

- **FunEx.html / FunEx.js**  
  Contains example functions and their invocation through the browser UI.

- **FuncProg.html / FuncProg.js**  
  Shows examples related to functional programming patterns using JavaScript in a web context.

All files are built with plain HTML, CSS (if present), and JavaScript, without frameworks, to reinforce fundamental web development skills.

---

## Technologies Used

- **HTML** – Structure of the web page and interface elements.  
- **JavaScript** – Core logic, functions, and interactivity in the browser.  
- **CSS (optional)** – Basic styling for visual layout (if any).  
- **Browser DevTools** – Used for inspection, debugging, and learning.
